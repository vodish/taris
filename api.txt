<h2>Список запросов API</h2>

<b>rtoken</b> - одноразовая подпись каждого запроса, меняется при использовании.

<pre class="white">
Входящие запросы на /api

-- список пользователей
    $_POST[userList]    =   1
    $_POST[wait]        =   [userList]

    user::list()

    $json[userList] = text


-- получить список пользователей

    $_POST[userGetCode] =   1
    $_POST[email]       =   email@email.com
    $_POST[wait]        =   [send, code]

    user::getCode()

    $json[send] = text
    $json[code] = 1234


-- проверить код
    $_POST[userGetCode] =   1
    $_POST[email]       =   email@email.com
    $_POST[code]        =   1234

    user::checkCode()
    

-- получить пачку

$_POST[pack]
    + $lineHtml
    + $lineText
    + $treeText
    + $accessText

pack::getPack()



user::checkCode()   -- список пользователей
user::checkCode()   -- список пользователей
user::checkCode()   -- список пользователей
</pre>
